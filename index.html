Mes Compétences

Développement Web

Acquisition des bases en HTML et CSS pour la création de sites web responsifs et modernes. Dans le cadre d'un projet de groupe, j'ai participé au développement d'un site web présentant l'entreprise Deezer, ce qui m'a permis de :

  • Travailler sur un projet structuré avec un cahier des charges détaillé définissant les pages requises (Histoire, À propos, Économie)
  • Apprendre à structurer le contenu avec HTML5 en suivant les bonnes pratiques de sémantique
  • Découvrir la mise en page responsive avec CSS3 pour assurer une expérience utilisateur adaptée sur tous les appareils
  • Créer un design moderne et esthétique en utilisant les fonctionnalités avancées de CSS (flexbox, grid, animations)
  • Expérimenter avec des animations et transitions pour améliorer l'interactivité du site
  • Découvrir le travail en équipe sur un projet web concret en respectant les contraintes du cahier des charges

Programmation

Apprentissage des concepts fondamentaux de programmation et des algorithmes. Dans le cadre d'une SAE, j'ai développé un pauvocodeur en Java, un outil de traitement sonore, ce qui m'a permis de :

  • Découvrir et implémenter des algorithmes de traitement du signal audio en Java
  • Apprendre les structures de données et les algorithmes fondamentaux
  • Développer une application avec une interface utilisateur en utilisant les bibliothèques Java
  • Explorer la manipulation et le traitement de fichiers audio avec les API Java
  • Comprendre les principes de la programmation orientée objet en Java

Bases de données

Découverte des concepts fondamentaux des bases de données relationnelles et de leur intégration dans des applications web. Dans le cadre d'une SAE, j'ai participé au développement d'un site e-commerce, ce qui m'a permis de :

  • Apprendre à concevoir et implémenter une base de données SQL pour gérer les articles
  • Développer des requêtes SQL pour la gestion des données produits
  • Explorer l'intégration de Python avec SQL pour créer une API REST
  • Découvrir la gestion de données pour le e-commerce
  • Comprendre l'importance de la cohérence des données
